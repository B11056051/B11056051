MDN - Mozilla

匯集眾多Mozilla基金會產品和網路技術開發文件的免費網站。

Codecademy: Learn to Code - for Free

Codecademy是一個線上交互式網站平台，它提供免費寫程式課程，其程式語言包括例如Python、JavaScript和Ruby，也包括置標語言例如HTML和CSS。

W3Schools Online Web Tutorials

在w3school，你可以找到你所需要的所有的網站建設教程。 從基礎的HTML到XHTML，乃至進階的XML、SQL、資料庫、多媒體和WAP。

Coderanch

CodeRanch 是一個學習與討論編程的友善論壇。我們曾經被稱為JavaRanch，但現在已經擴展了其他語言。您可以詢問有關 HTML、JavaScript、CSS、Python、Groovy 等許多其他語言的問題。

freeCodeCamp

freeCodeCamp是一個非營利性組織，由交互式學習Web平台，在線社區論壇，聊天室，在線出版物和旨在使所有人均可訪問學習Web開發的本地組織組成。從向學生介紹HTML，CSS和JavaScript的教程開始，學生將逐步完成他們單獨或成對完成的項目任務。

TMU程式自學平台 - 臺北醫學大學

Stack Overflow

Stack Overflow是一個程式設計領域的問答網站，隸屬Stack Exchange Network。網站允許註冊使用者提出或回答問題，利用積分制度讓使用者自行管理。

iT 邦幫忙::一起幫忙解決難題，拯救IT 人的一天

iT 邦幫忙是IT 領域的技術問答與分享社群。

DEV Community

Dev.to是一個程式開發人員互相交流的平台，開發人員也在此分享他們的研究成果。

SitePoint

通過SitePoint的教程、課程和書籍來學習網頁設計開發-HTML、CSS、JavaScript、PHP、響應式網頁設計等。

CodeProject - For those who code

The Code Project，是一個免費公開來源碼的程式設計網站，主要的使用者是Windows平台上的電腦程式設計人員。每一篇文章幾乎都附有來源碼和例子下載。

LeetCode

LeetCode是一個收集軟體工程師面試考古題的線上練功網站。

Codewars

Codewars是計算機編程的教育社區。在該平台上，軟件開發人員接受稱為kata的編程挑戰的培訓。這些離散的編程練習以各種編程語言培訓了一系列技能，並在在線集成開發環境中完成。在Codewars中，社區和挑戰的進展是遊戲化的，用戶由於完成kata，貢獻kata和高質量的解決方案而獲得了等級和榮譽。

Codeforces

Codeforces是一個提供線上解題系統的俄羅斯網站。該網站由一群來自itmo大學的程式設計師建立並維護，其中主要的領導者為Mikhail Mirzayanov。自2013年起，Codeforces的活躍使用者數就已經超過了另一大線上解題系統網站TopCoder。

HackerRank

HackerRank 是一個專屬於工程師的網站，提供了許多「練習題」，你可以純粹用來練習自己的coding 技巧，也可以藉著完成越來越多的題目，展現自己的實力。

CodeSignal

CodeSignal也被稱為開發者評估平台，它提供了一個基於雲技術技能篩選軟件，由專業設計的技能評估和專有的預測性編碼得分提供支持。

Online Judge

UVa線上解題系統是由西班牙瓦雅多利大學開發的自動化線上程式評判系統，包含了超過4300個程式設計問題並且開放給所有人註冊參與解題。
